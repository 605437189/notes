        新的type : email, url, number, date, range ,color  属性： placeholder , autofocus, required
	3.多媒体：
	    video, audio
	    API: play, pause, timeupdate等
	4.离线存储
		 localStorage:长久存储，大小约5mb 
		 sessionStorage：会话存储，浏览器窗口关闭就消失
	 5.通信与性能
		 websocket:全双工通信
		 web workers:在后台运行JS， 不阻塞主线程
		 canvas/svg:绘图技术

5.http status code
	
	101 Switching Protocols   websocket
	200 success
	204 no content  如DELETE请求
	301 move permanently 永久重定向
	302 found 临时重定向
	304 not modified 未修改
	400 bad request 参数不对
	401 unauthorized 未授权 没登录或 token 过期
	403 forbidden 没有权限
	404 not found
	405 method not allowed
	500 internal server error
	502 bad gateway 网关错误
	504 gateway timeout
